# 伯克希尔·哈撒韦

## 公司简介

伯克希尔·哈撒韦是 [[沃伦·巴菲特]] 长期经营的控股公司，也是巴菲特投资思想最完整的实践场。

## 投资故事

公司从纺织企业转型为保险、能源、铁路、消费品和证券投资组合的复合体。其资本配置体现了 [[能力圈]]、[[安全边际]] 和长期主义。
